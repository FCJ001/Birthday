# 加加生日快乐（微信可打开 H5）

这是一个**可直接在微信里打开的浪漫风生日祝福网页**（H5）。打开后先点一下解锁音乐，然后会出现心形/烟花等动画与祝福文案。

## 使用方式

### 1) 放入音乐

把你想要的背景音乐（mp3）放到：

- `assets/music.mp3`

你也可以替换成自己的文件名，但需要同步修改 `index.html` 里 `<audio>` 的 `src`。

### 2) 本地预览

最稳的方式是起一个本地静态服务器（避免部分浏览器的音频/资源限制）。

如果你电脑装了 Node.js：

```bash
npx serve .
```

然后用提示的地址在手机浏览器/微信里打开。

### 3) 部署到线上（发微信链接）

把整个目录部署到任何**支持 HTTPS** 的静态托管（例如 Vercel / Netlify / GitHub Pages / 云存储静态网站）。

部署完成后把链接发到微信即可打开；也可以用任意工具把链接转成二维码。

## 小提示（微信音频规则）

微信通常需要用户点击一次页面后，音乐才允许播放，这是正常现象。页面的「点一下，开启惊喜」就是为了解锁音乐与动效。
